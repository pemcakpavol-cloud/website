O aplikácii DocuMind

DocuMind je študentský projekt na správu dokumentov s využitím umelé inteligencie. Aplikácia umožňuje nahrávať dokumenty, prevádzať ich na text pomocou OCR a ukladať dôležité poznámky. Podrobnosti o tom, ako funguje backend a frontend, nájdete nižšie.

Backend – Python (FastAPI)

FastAPI

Webový framework v Pythone, ktorý spracováva logiku aplikácie, databázu a vytvára API endpointy. Backend komunikuje s frontendom cez HTTP volania.

Uvicorn

Server pre FastAPI. Spúšťa backend a zabezpečuje, že API je dostupné pre frontend a aplikáciu.

Tesseract OCR

Knižnica na rozpoznávanie textu z obrázkov. Používame ju na prevod faktúr, poznámok a dokumentov do čistého textu.

SQLite (lifedocs.db)

Jednoduchá databáza, ideálna pre školský projekt. Ukladáme názov súboru, text z OCR, poznámky a dátum vytvorenia.

SQLAlchemy

ORM knižnica pre prácu s databázou ako s objektami, zjednodušuje manipuláciu s dátami.

Frontend – React / React Native

React / React Native

Zobrazuje používateľské rozhranie (UI), umožňuje nahrať súbor, zobraziť text a uložiť dokument.

Fetch / API volania

Frontend komunikuje s backendom cez HTTP a posiela JSON dáta. Zabezpečuje správnu synchronizáciu dát.
